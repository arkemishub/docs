# Frontend

Here you'll find all resources for Arke Frontend, if you're starting with Arke frontend development we suggest to check
the pre-requisite knowledge, if you're already used Arke and are searching specific docs use the search bar on the
right corner of the docs.

### Pre-Requisite Knowledge
We need to establish a baseline so that the docs can stay focused on Arke Frontend.
It's recommended that you have a intermediate understanding of React and NextJS framework.

If you need to brush up on your skills, check out the following links, which will introduce you to the fundamentals:

| Topic        | Description |
| :------------ | :---------: |
| [React](https://react.dev/learn) | React is a powerful JavaScript library for building interactive user interfaces and dynamic web applications.  |
| [Next.js](https://nextjs.org/docs)    | Next.js is a React framework for building full-stack web applications. You use React Components to build user interfaces, and Next.js for additional features and optimizations. |
| [TailwindCSS](https://tailwindcss.com/docs) | Tailwind CSS is a utility-first CSS framework that empowers developers to build modern, responsive web interfaces with unparalleled flexibility and efficiency. |
| [Arke Frontend Architecture](/docs/think-arke/architecture/frontend) | High-Level Arke Architecture schema and concepts to understand the framework

If you're ready to start goes on the next chapter to discover how to set a Arke Frontend project.
