# Architecture
Our goal is distribute our application for our end-users. On below you can see a live production application developed
with Arke.

![ArkeDeployedApplication](/architecture/architecture_deployed_1.png)

In Arke each services as backend, frontend and console are Docker containerized applications. This approach makes it
Cloud Provider agnostic, so we can deploy them on common platform as Google Cloud Platform, Amazon Web Service, Azure etc.

Continue on architecture section to discover the specifics of different services.
