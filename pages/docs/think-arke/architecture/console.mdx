### Add a Database client to manage your server
To view and manage your Server database you'll need a PostgreSQL client.

There are different options, below you can find a licensed and open solutions:
- [Datagrip](https://www.jetbrains.com/datagrip/download/#section=mac): JetBrains DataGrip is a cross-platform IDE
for working with SQL and databases. DataGrip includes an evaluation license key for a free 30-day trial.
- [PGAdmin](https://www.pgadmin.org/): pgAdmin is the most popular and feature rich Open Source administration and
development platform for PostgreSQL
