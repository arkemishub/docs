CLI usage

arkectl is the official CLI for Arke. It is a command-line tool that allows you to interact with Arke from a terminal window. It allows you to create new projects, start the console and the API service.

Prerequisites​

  • Docker
  • Homebrew or GO

Installation​

  1. Install arkectl using GO.
    go install github.com/arkemishub/arkectl@latest
  2. Expose a new variable called ARKEPATH within your .zshrc or .bashrc
    .zshrc
    export ARKEPATH=$HOME/arke
  3. Run arkectl install to download required files to run Arke locally.
    arkectl install

Commands​

install​

Downloads required files to run Arke locally.

arkectl install

create-app​

Bootstrap Arke environment by starting the console and the API service.

Args:

  • project-name - Name of the project to create (optional)
arkectl create-app <project-name>

start​

Starts the console and the API service.

arkectl start

update​

Updated console and API service docker images to the latest version.

arkectl update

Notes​

By using the CLI you will be able to create a new project, start the console and the API service. By default the console will be available at http://localhost:3100 and the API service at http://localhost:4000.
