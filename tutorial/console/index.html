Structure your data

In order to structure your data, you will use the Arke console.

Log in to the consoleâ€‹

Once the console is started, you need to log in. By default, you can use:

username: admin
password: admin

Since you ran arkectl create-app nextjs-todo you don't need to create a project. Just select it from the dropdown of get started page as shown below.

Create your first Arkeâ€‹

Since our goal is to create a todo app we need to create an Arke named Todo.

  • Navigate to http://localhost:3100/arke
  • Click Add Arke
  • Fill the form as following

Create parametersâ€‹

For our todo list, we will create following parameters: title, done and priority. It's important to understand which type you need for each parameter, in our case title will be a string, done will be a bool and priority an enum.

  • Navigate to http://localhost:3100/parameters
  • Click Add Parameter
  • Create the parameters as shown below.

Link parametersâ€‹

Now that we have the parameters all we have to do is link them to our Todo Arke.

  • Navigate to http://localhost:3100/arke/todo
  • Click on Parameters tab
  • Click Link Parameters
  • Link the parameters as shown below

Create a todoâ€‹

Let's create our first task:

  • Navigate to http://localhost:3100/arke/todo
  • Click Add Todo
  • Fill the form with some data

Recapâ€‹

Congrats, you created your first Arke!
